# rawmem 0.5.0

Version 0.5.0 makes installation and removal inspectable on the maintained
Windows platform.

## Added

- `rawmem doctor` checks config, ledger writability, daemon freshness,
  localhost health, token authentication, startup registration, global Git
  hooks, and recent events.
- `rawmem setup --dry-run` previews project and machine changes without writes
  or confirmation flags.
- `rawmem uninstall` stops and removes the Windows startup task, unsets rawmem's
  global Git hook path, and removes the rawmem PowerShell profile block.
- `rawmem uninstall --remove-home --yes` explicitly deletes local config, state,
  generated integrations, and the append-only ledger.
- The browser extension options page can test daemon connectivity and token
  acceptance.
- Windows CI covers Python 3.10 and 3.12, compilation, unit tests, the
  open-source audit, and a CLI smoke test.

## Safety

- Uninstall preserves `~/.rawmem` by default.
- Diagnostic output never prints the browser token.
- Browser token testing uses a dedicated authenticated endpoint that exposes no
  local storage path.
- Linux startup behavior is not claimed or tested in this release.
